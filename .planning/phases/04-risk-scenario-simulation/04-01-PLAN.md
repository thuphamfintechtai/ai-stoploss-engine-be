    VaR Service (varService.js):
    - calculateHistoricalVaR({ positions, ohlcvBySymbol, confidenceLevel=0.95, lookbackDays=60 })
    - Input: positions = [{ symbol, entry_price, quantity, exchange }], ohlcvBySymbol = { VCB: [{close:...},...] }
    - Per D-01: Historical Simulation = sort daily returns, lay percentile (1 - confidenceLevel)
    - Per D-02: Dung simple-statistics percentile, mean, stddev
    - Per D-03: Tra ve { portfolioVaR: { varVnd, varPercent, confidenceLevel }, positionVaRs: [{ symbol, varVnd, varPercent }] }
    - Per D-04: Kem sentence: "Voi 95% tin cay, max loss 1 ngay la X VND (Y% portfolio)"
    - Edge case: Neu ohlcv < 20 ngay cho mot symbol -> warning, van tinh voi du lieu co
    - Edge case: Neu portfolio rong -> return varVnd=0

    Monte Carlo Service (monteCarloService.js):
    - runMonteCarloSimulation({ positions, ohlcvBySymbol, numPaths=1000, numDays=20, exchangeBySymbol={} })
    - Per D-05: GBM: dS = mu*S*dt + sigma*S*dW. mu = mean(daily_returns), sigma = stddev(daily_returns)
    - Per D-06: 1000 paths, 20 trading days
    - Per D-07: Clamp moi ngay: HOSE +/-7%, HNX +/-10%. exchangeBySymbol = { VCB: 'HOSE' }
    - Per D-08: Output: { percentileBands: { p5, p25, p50, p75, p95 } (mang 20 gia tri moi band), probabilityOfLoss: number, paths: number[][] (giu nguyen 1000 paths cho chart) }
    - Per D-09: Server-side, khong can Web Workers
    - Random: dung Math.random() + Box-Muller transform cho normal distribution
    - Tinh portfolio value moi ngay = sum(position_value_i * simulated_return_i)

    Stress Test Service (stressTestService.js):
    - calculateStressTest({ positions, scenarios=[-10,-15,-20], customScenario=null })
    - Per D-10: 3 predefined scenarios
    - Per D-11: impact = position_value * beta * (scenario_drop / 100)
    - Per D-12: SECTOR_BETAS = { BANKING: 1.2, REAL_ESTATE: 1.5, TECHNOLOGY: 0.8, STEEL: 1.3, SECURITIES: 1.4, ENERGY: 0.9, CONSUMER: 0.7, RETAIL: 0.8, OTHER: 1.0 }
    - Per D-13: Custom scenario support -- user nhap % drop rieng
    - Dung getSector() tu sectorClassification.js de map symbol -> sector -> beta
    - Output: { scenarios: [{ dropPercent, totalImpactVnd, totalImpactPercent, positions: [{ symbol, sector, beta, positionValue, impactVnd, impactPercent }] }] }
  </behavior>
  <action>
    TDD flow: Viet test truoc, implement sau.

    **varService.js:**
    - Export function calculateHistoricalVaR(params)
    - Tinh daily returns tu ohlcv close prices: return_i = (close_i - close_{i-1}) / close_{i-1}
    - Portfolio daily return = weighted sum theo position value
    - Sort returns, lay percentile(returns, 1 - confidenceLevel) -> VaR percent
    - VaR VND = totalPortfolioValue * abs(varPercent)
    - Per-position VaR: rieng tung symbol
    - Return object voi summary sentence per D-04

    **monteCarloService.js:**
    - Helper: boxMullerRandom() -> standard normal
    - Tinh mu, sigma tu historical returns cho moi symbol
    - Loop 1000 paths x 20 days: portfolioValue[day] = sum(position_i * simPrice_i)
    - simPrice = prevPrice * exp((mu - sigma^2/2)*dt + sigma*sqrt(dt)*Z) -- GBM
    - Clamp: maxDailyChange = exchange === 'HNX' ? 0.10 : 0.07
    - After 1000 paths: tinh percentile bands cho moi ngay
    - probabilityOfLoss = count(finalValue < initialValue) / numPaths

    **stressTestService.js:**
    - SECTOR_BETAS constant (per D-12)
    - Loop scenarios, loop positions: impact = qty * entry_price * beta * dropPercent/100
    - Custom scenario: them vao scenarios array
    - Return structured result

    **KHONG import database** -- tat ca input la parameters. Controller se fetch tu DB roi pass vao.
