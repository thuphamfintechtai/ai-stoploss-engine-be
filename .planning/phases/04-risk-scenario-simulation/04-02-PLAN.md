    Them 4 Joi schemas (dat gan cac schemas hien co):

    **varSchema:** portfolio_id: Joi.string().uuid().required(), confidence_level: Joi.number().min(0.9).max(0.99).default(0.95), lookback_days: Joi.number().integer().min(20).max(250).default(60)

    **monteCarloSchema:** portfolio_id: Joi.string().uuid().required(), num_paths: Joi.number().integer().min(100).max(5000).default(1000), num_days: Joi.number().integer().min(5).max(60).default(20)

    **stressTestSchema:** portfolio_id: Joi.string().uuid().required(), custom_scenario: Joi.number().min(-50).max(0).optional()

    **sectorConcentrationSchema:** portfolio_id: Joi.string().uuid().required()

    Them 4 controller functions (cuoi file, truoc export neu co):

    **getVaR(req, res, next):**
    - Validate req.query voi varSchema
    - Query positions: `SELECT symbol, entry_price, quantity, exchange FROM positions WHERE portfolio_id = $1 AND status = 'OPEN'` (dung DB_SCHEMA)
    - Neu khong co positions -> return { success: true, data: { portfolioVaR: { varVnd: 0, varPercent: 0 }, positionVaRs: [] } }
    - Fetch OHLCV cho moi symbol (dung fetchOHLCV existing, limit = lookback_days + 10 de du buffer): Promise.all, gioi han 5 concurrent (giong dynamicSL pattern)
    - Build ohlcvBySymbol map
    - Goi calculateHistoricalVaR({ positions, ohlcvBySymbol, confidenceLevel, lookbackDays })
    - Return res.json({ success: true, data: result })

    **getMonteCarloSimulation(req, res, next):**
    - Validate req.body voi monteCarloSchema (POST vi co the heavy)
    - Query positions giong getVaR
    - Fetch OHLCV 250 ngay cho moi symbol (can nhieu data hon cho mu, sigma chính xác)
    - Build exchangeBySymbol tu positions
    - Goi runMonteCarloSimulation
    - **KHONG tra 1000 paths raw** -- chi tra percentileBands + probabilityOfLoss + initialValue + finalValueDistribution (summary). Frontend se ve chart tu percentileBands.
    - Return res.json({ success: true, data: result })

    **getStressTestResult(req, res, next):**
    - Validate req.body voi stressTestSchema (POST)
    - Query positions giong getVaR
    - Build positions array voi entry_price, quantity, symbol
    - Goi calculateStressTest({ positions, customScenario: value.custom_scenario })
    - Return res.json({ success: true, data: result })

    **getSectorConcentrationResult(req, res, next):**
    - Validate req.query voi sectorConcentrationSchema (GET)
    - Query positions giong getVaR
    - Goi calculateSectorConcentration({ positions })
    - Return res.json({ success: true, data: result })

    Concurrency pattern cho OHLCV fetch (tranh rate limit):
    ```javascript
    // Fetch OHLCV 5 concurrent max
    const symbols = [...new Set(positions.map(p => p.symbol))];
    const ohlcvBySymbol = {};
    for (let i = 0; i < symbols.length; i += 5) {
      const batch = symbols.slice(i, i + 5);
      const results = await Promise.all(batch.map(sym => fetchOHLCV(sym, exchangeMap[sym] || 'HOSE', limit)));
      batch.forEach((sym, idx) => { ohlcvBySymbol[sym] = results[idx]; });
    }
    ```
